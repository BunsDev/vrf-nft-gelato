# Gelato-NFT

Template project using Gelato VRF to generate randomized NFTs.

## Setup

1. Clone the repo
2. Install forge modules `forge install`
3. Build the project: `forge build`

## Test

Run `forge test -vvv`

## Deploy

`forge script ./script/DeployIceCreamNFT.s.sol --rpc-url ${rpcurl}`
